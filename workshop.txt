description=[h1]Config (SP / Host / Dedicated)[/h1]
description=[b]Singleplayer / Host:[/b] when creating a world, open the [b]Woodcutting[/b] tab in Sandbox Options and adjust.
description=
description=[b]Dedicated server:[/b] edit the save’s Sandbox (or create a preset) and set:
description=[code]
description=SandboxVars.Woodcutting = {
description=  damageBaseMultiplier = 1.0,
description=  damagePerLevel       = 0.15,
description=  damageMaxMultiplier  = 8.0,
description=  oneHitLevelThreshold = 6,
description=  oneHitTreeDamage     = 2000,
description=  onlyForAxes          = true,
description=  xpMultiplier         = 1.0,
description=
description=  cumulatedForagingAndWoodcuttingSkillLevelForFruit = 8,
description=  FruitTreeExtra = 80,
description=  Winter         = 130,
description=  Pinecone       = 20,
description=  PineTreeExtra  = 120,
description=  Log            = 40,
description=  TreeBranch     = 35,
description=  Twigs          = 30,
description=}
description=[/code]
description=[i]Tip:[/i] with the defaults, around level [b]6–7[/b] you can one-hit trees if the threshold is enabled.
description=
