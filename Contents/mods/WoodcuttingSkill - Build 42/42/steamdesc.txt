[h1]Woodcutting Skill — Overhaul[/h1]
[i]“The Apocalypse has come… and TREES are the real threat.”[/i]

This is an [b]overhaul[/b] of the original mod “[i]Woodcutting Skill[/i]” by [b]Champy[/b].  
Original mod: [url=https://steamcommunity.com/sharedfiles/filedetails/?id=2944004910]https://steamcommunity.com/sharedfiles/filedetails/?id=2944004910[/url]  
All credit for the original idea and baseline implementation goes to Champy. This version rewrites and extends several systems (damage scaling, XP, extra loot, and full Sandbox configuration), with server-authoritative logic for MP.

[hr][/hr]

[h1]What it does[/h1]
Adds a dedicated [b]Woodcutting[/b] skill that makes chopping trees and removing bushes easier and more rewarding.

[b]Vanilla-style benefits (kept/improved):[/b]
[list]
[*]Less endurance used (−7% per level)
[*]Less calories used (−12% modifier every 3 levels)
[*]Lower chance to lose axe condition (when chopping trees)
[*]Flat axe tree damage bonus (+2 per level)
[*]Chance of extra items on medium/large trees
[*]Removing bushes: −7% endurance per level and +10% action speed per level
[*]At Woodcutting 8, chopping won’t escalate to “Excessive Exertion” (still “High Exertion”)
[/list]

[b]Extra items (rare at low levels):[/b]
[list]
[*]Logs, Tree Branches, Twigs
[*]Pinecones (pine-like trees)
[*][spoiler]Tree fruits & acorns (if [Woodcutting + Foraging] ≥ 8 and it’s not winter)[/spoiler]
[*][spoiler]Dead squirrels (very rare; skill-gated)[/spoiler]
[/list]

[i]Nature Abundance[/i] affects those chances (−20%, −10%, +10%, +20%).  
Small trees (size 1–2) never grant extras.

[hr][/hr]

[h1]What’s NEW in this Overhaul[/h1]
[list]
[*][b]Damage per Level (scaling) + One-Hit Threshold[/b]: tree damage now scales with your Woodcutting level. By default, at [b]level 6[/b] you reach a configurable [b]1-hit[/b] threshold.
[*][b]Full Sandbox Page (SP & MP)[/b]: tweak everything in Custom Sandbox or via Admin Panel (MP). No file editing needed.
[*][b]XP Multiplier (Sandbox)[/b]: globally increase/decrease Woodcutting XP gained per hit.
[*][b]Extra Looting logic reworked[/b]: chances scale with [i](Woodcutting + Foraging)[/i] and respect Nature Abundance. Loot is granted reliably (server-side) to the player’s inventory.
[*][b]Server-authoritative logic[/b]: damage scaling and extras run on the server for consistent multiplayer behavior.
[*][b]Translations scaffold[/b]: English + PT-BR keys included (feel free to contribute more).
[/list]

[hr][/hr]

[h1]Sandbox Settings (SP & MP)[/h1]
This overhaul adds a full Sandbox page: [b]Woodcutting[/b]. You can configure:

[list]
[*][b]Base Damage Multiplier[/b] — multiplier at level 0, before per-level scaling.
[*][b]Damage per Level (+%)[/b] — e.g., 0.15 = +15%/level.
[*][b]Max Damage Multiplier (cap)[/b] — upper limit for total scaling.
[*][b]One-Hit Level Threshold[/b] — at/above this level, trees fall in 1 hit. (Set to 99 to disable.)
[*][b]One-Hit TreeDamage value[/b] — TreeDamage used once threshold is reached.
[*][b]Affect only Axes[/b] — limit scaling to axes only.
[*][b]XP Multiplier[/b] — global Woodcutting XP multiplier on tree hits.
[*][b]Extra Loot (1 in N)[/b] — Log, Branch, Twigs, Pinecone, Pine Extra, Winter, Fruit Tree.
[*][b]Min (Foraging + Woodcutting) for Fruit[/b] — minimum combined levels for fruit extras.
[/list]

[b]Single-player (new world):[/b] New Game → [i]Custom Sandbox[/i] → [b]Woodcutting[/b].  
[b]Single-player (existing world):[/b] host as MP and use Admin Panel, or edit the save’s SandboxVars.lua.  
[b]Multiplayer (host):[/b] ESC → Admin Panel → Sandbox Options → [b]Woodcutting[/b] → Apply (restart recommended).  
[b]Dedicated server:[/b] edit `<ServerName>_SandboxVars.lua`:

[code]
SandboxVars = SandboxVars or {}
SandboxVars.Woodcutting = SandboxVars.Woodcutting or {}
local W = SandboxVars.Woodcutting
W.damageBaseMultiplier = 1.0
W.damagePerLevel = 0.15
W.damageMaxMultiplier = 8.0
W.oneHitLevelThreshold = 6
W.oneHitTreeDamage = 2000
W.onlyForAxes = true
W.xpMultiplier = 1.0
W.cumulatedForagingAndWoodcuttingSkillLevelForFruit = 8
W.FruitTreeExtra = 80
W.Winter = 130
W.Pinecone = 20
W.PineTreeExtra = 120
W.Log = 40
W.TreeBranch = 35
W.Twigs = 30
[/code]

[i]Tip:[/i] set [b]oneHitLevelThreshold[/b] to 99 to disable the 1-hit feature.

[h1]Traits & Professions[/h1]
Adds a [b]Woodcutter[/b] trait (+1 Woodcutting for 2 points).  
Also buffs:
[list]
[*]Lumberjack occupation: +3 Woodcutting
[*]Park Ranger occupation: +2 Woodcutting
[*]Farmer occupation: +1 Woodcutting
[*]Gardener trait: +1 Woodcutting
[/list]

There are currently [u]no skill books[/u] for Woodcutting.

[h1]Compatibility[/h1]
This mod overrides the [b]Chop Tree[/b] and [b]Remove Bush[/b] timed actions.  
If another mod touches the same actions, put this one [i]after[/i] in load order.  
Known to work with most setups; server-authoritative logic helps MP consistency.
